# Severina
Telegram chatbot

## Install repository


**Create an Virtual env**

`sudo pip install virtualenv`

`virtualenv [opções] <nome_da_pasta>`

`source nome_da_pasta/bin/activate`

`python --version`

Deactivate
`$ deactivate`

Remove
`$ rm -r nome_da_pasta`

**Add your Virtual env (nome_da_pasta) to the .gitignore file**

**Install Telepot**
`$ pip install telepot`
